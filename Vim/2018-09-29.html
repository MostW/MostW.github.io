# 在Mac下vim如何使用系统剪切板

Mac OS自带的vim不支持clipboard，在终端输入

vim --version

回车，可以发现vim的clipboard项前面是-，也就是不支持该功能。

因此，可以在终端通过homebrew安装新的vim

brew install vim

安装的时候，发现每次用brew的命令后，首先会出现的是

homebrew update···

这个过程浪费了很多时间，所以设置禁止homebrew自动更新，将下面的代码写到~目录下面的.zshrc文件（如果没有就创建一个),然后source .zshrc文件即可

# brew 不自动更新
export HOMEBREW_NO_AUTO_UPDATE=true

装好vim之后，就可以使用 "+y 这三个键复制内容到剪切板， "+p 这三个键粘贴系统剪切板内容到vim
